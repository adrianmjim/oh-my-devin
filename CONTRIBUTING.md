# Contributing to Oh My Devin

## Development setup

See the [README](README.md#development) for install, test, and build commands.
Pull requests are expected to keep the CI suite green: lint, format, typecheck,
tests, and build.

## Contributor License Agreement

Every contributor must sign the
[Individual Contributor License Agreement](CLA.md) before their first pull
request can be merged. Signing is automated: the CLA bot comments on your first
pull request with instructions, and you sign by replying with the exact
sentence it requests. One signature covers all your past and future
contributions to the Project.
